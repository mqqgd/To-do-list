### [67\. 二进制求和](https://leetcode-cn.com/problems/add-binary/)

Difficulty: **简单**


给你两个二进制字符串，返回它们的和（用二进制表示）。

输入为 **非空** 字符串且只包含数字 `1` 和 `0`。

**示例 1:**

```
输入: a = "11", b = "1"
输出: "100"
```

**示例 2:**

```
输入: a = "1010", b = "1011"
输出: "10101"
```

**提示：**

*   每个字符串仅由字符 `'0'` 或 `'1'` 组成。
*   `1 <= a.length, b.length <= 10^4`
*   字符串如果不是 `"0"` ，就都不含前导零。


#### Solution

Language: ****

```
/**
 * @param {string} a
 * @param {string} b
 * @return {string}
 */
var addBinary = function(a, b) {
    let res = '';
    let i1 = a.length - 1,
        i2 = b.length - 1;
    let carry = 0;
    while (i1 >= 0 || i2 >= 0) {
        const x = i1 >= 0 ? a[i1] - '0' : 0;
        const y = i2 >= 0 ? b[i2] - '0' : 0;

        const sum = x + y + carry;
        res += (sum % 2);
        carry = Math.floor(sum / 2);

        i1--;
        i2--;
    }
    if (carry) res += carry;
    return res.split("").reverse().join("");
};
```
